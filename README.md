# 🎮 Video Game Sales Analysis using SQL

## 📌 Overview
This project focuses on analyzing a video game sales dataset using SQL.  
The analysis is performed by solving real-world business questions and extracting insights directly from the data.

---

## 🎯 Objectives
- Analyze global video game sales performance
- Identify top-selling games
- Compare regional sales (NA, JP, PAL)
- Evaluate console/platform performance
- Understand genre popularity
- Study sales trends over time
- Analyze publisher performance

---

## 📊 Analysis Performed
The project includes SQL queries to answer the following:

1. Total global sales
2. Top 10 best-selling games
3. Regional sales contribution
4. Top games by region
5. Console performance analysis
6. Average sales per console
7. Genre-wise sales analysis
8. Sales trends over time
9. Top publishers by sales

---

## 📸 Approach
Each analysis includes:
- SQL query
- Output/result screenshot

This project focuses on SQL-based analysis rather than visualization.

---

## 🛠️ Tools Used
- SQL (MySQL)
- Canva
